
# EMS Mobile - PWA Version

Aplikasi mobile untuk staff lapangan, dibuat sebagai Progressive Web App (PWA) sebagai alternatif dari Flutter.

## Fitur
- Login
- List project yang ditugaskan
- Detail project dan item
- Upload foto ke Google Drive
- Status foto

## Cara Menggunakan

### 1. Buka Aplikasi di Browser Ponsel
Buka URL aplikasi (misal `https://your-domain.com/mobile`) di browser ponsel (Chrome/Safari).

### 2. Instal sebagai Aplikasi PWA
- **Chrome**: Tap menu titik tiga → "Add to Home Screen"
- **Safari**: Tap share → "Add to Home Screen"

Aplikasi akan muncul di layar utama ponsel seperti aplikasi native!

### 3. Login
Masukkan email dan password yang sudah dibuat oleh admin.

## Teknologi
- Next.js App Router
- Tailwind CSS
- Supabase
- Google Drive API
